SakeShopDetailsContent

@Composable
fun SharedTransitionScope.SakeShopDetailsContent(sakeShop: SakeShop, modifier: Modifier = Modifier, onMapsClick: (String) -> Unit = {}, onWebsiteClick: (String) -> Unit = {}, animatedVisibilityScope: AnimatedVisibilityScope, onBackClick: () -> Unit)

Composable function that displays detailed content for a SakeShop within a shared transition scope.

This layout includes:

  • A header image with shared transition effect.

  • Sake shop name with shared transition effect.

  • Rating display with star icon.

  • Description of the sake shop.

  • Address section with location icon and coordinates.

  • Action buttons for opening location in maps and visiting the website.

Parameters

sakeShop

The data model representing the sake shop details to display.

modifier

Optional Modifier for styling this composable.

onMapsClick

Lambda called with the Google Maps URL when the "Open in Maps" button is clicked.

onWebsiteClick

Lambda called with the website URL when the "Visit Website" button is clicked.

animatedVisibilityScope

The AnimatedVisibilityScope used for shared transition animation.

onBackClick

Lambda called when a back action is triggered (not used in the current UI).
