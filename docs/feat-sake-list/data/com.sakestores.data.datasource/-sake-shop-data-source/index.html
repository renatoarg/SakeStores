SakeShopDataSource

class SakeShopDataSource @Inject constructor(context: Context, gson: Gson)

Data source responsible for loading sake shop data from local assets.

This class uses Gson to parse JSON data stored in the app's assets folder. It provides suspend functions to asynchronously fetch the list of sake shops or retrieve a specific sake shop by its name.
