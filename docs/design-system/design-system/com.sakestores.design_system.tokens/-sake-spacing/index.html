SakeSpacing

object SakeSpacing

Defines standard spacing values used throughout the Sake Stores design system.

Spacing values are represented as dp units and help maintain consistent padding, margins, and gaps across the UI.

Includes predefined sizes from extra small (4.dp) to triple extra large (64.dp), as well as semantic aliases for common layout needs such as card padding, screen padding, item spacing, and section spacing.
